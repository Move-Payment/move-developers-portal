title: "Step 1: Set Up the MovePay Plugin in Shopware"
---

Follow these steps to install the MovePay plugin in your Shopware store:

1. Log in to the **Shopware Admin Panel**.
2. Navigate to **Extensions â†’ My Extensions**.
3. Click **Upload Extension**.
4. Select the MovePay plugin `.zip` file from your computer and upload it.
5. After the upload completes, click **Install now** to activate the plugin.

Once the installation finishes, the MovePay plugin becomes available in your Shopware extensions list and is ready for configuration.
