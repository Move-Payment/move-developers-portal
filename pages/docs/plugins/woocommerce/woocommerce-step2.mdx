---
title: "Step 2: Set Up Payment Gateway Credentials"
---

To set up the payment gateway, you’ll need API credentials from your MovePay Merchant App. Follow the steps below:

### Retrieve API Key from MovePay Merchant App:
1. Log into your MovePay Merchant App.
2. Navigate to **Other** -> **API settings** tab.  
   ![API Settings](/api-settings.png)
3. In the **API Settings** section, you can manage your API tokens.
4. Click **Add API key** to create a new API key and copy it for use in your integration.  
   ![Generate API Token](/generate-api-token.png)

### Configure Plugin with API Token:
1. In the plugin settings, paste the API token you copied into the **API Token** field.

### Select Business:
1. In the MovePay Merchant App, navigate to **Manage Business**.
![Manage Business](/plugins/manage-business.png)
2. Select your business and copy the **Move ID**.
3. Paste the Move ID into the plugin’s **business ID** field.
![Filled Settings Example](/plugins/woocommerce/settings-filled.png)

### Enable the Plugin:
1. After entering the necessary credentials, ensure the **Enable MovePay Payment Gateway** option is checked.
2. Click **Save Changes** to finalize your configuration.
![Enabled Plugin Settings](/plugins/woocommerce/settings-enabled.png)
