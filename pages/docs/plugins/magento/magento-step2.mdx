---
title: "Step 2: Set Up Payment Gateway Credentials"
---

To successfully configure the payment gateway, you will need to retrieve API credentials from your MovePay Merchant App. Follow the steps below:

### Retrieve API Key from MovePay Merchant App

1. Log into your MovePay Merchant App.
2. Navigate to **Settings** and click on the **API** tab.
![API Settings](/plugins/settings.png)
3. Navigate to **API Settings**.
4. Generate a new API Token and copy it.
![Generate API Token](/plugins/generate.png)

### Configure Plugin with API Token

1. Open the plugin settings in Magento as per the steps outlined above.
2. Paste the copied API token into the **API Token** field of the plugin configuration.
![Plugin Settings Key](/plugins/magento/plugin-settings-key.png)

### Select Business

1. From the MovePay Merchant App, navigate to **Manage Business**.
![Manage Business](/plugins/manage-business.png)
2. Select your business and copy the **Move ID**.
3. Paste the Move ID into the pluginâ€™s **Business ID** field under settings.
![Plugin Settings Business](/plugins/magento/plugin-settings-id.png)

### Save Configuration

Click **Save Config** to save your changes.
